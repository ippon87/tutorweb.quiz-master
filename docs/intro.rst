Quick Start
===========

Start hacking away on this project by running::


Babel, first time::

- bin/py setup.py extract_messages
- bin/py setup.py init_catalog -l sl
- bin/py setup.py update_catalog
- bin/py setup.py compile_catalog

all next runs::

- bin/py setup.py extract_messages
- bin/py setup.py update_catalog
- bin/py setup.py compile_catalog
